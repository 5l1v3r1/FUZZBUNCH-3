# FUZZBUNCH

### Prerequisites
- `OS`: tested on Windows 7 x86 
- `Python 2.6.6`
- `Pywin32`
- `Java SE 8`


### Setup

- `Install all prerequisites` 
- `Run Fuzzbunch`: python fb.py'
